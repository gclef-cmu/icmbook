# Exercises

```{exercise}
:label: ex-13-echo
Implement a feedback echo. For a feedback gain of $g = 0.6$, roughly how many
echoes are audible before they fade below hearing?
```

```{exercise}
:label: ex-13-flanger
Build a flanger by modulating a very short delay with an LFO. How do LFO rate
and depth change the effect?
```

```{exercise}
:label: ex-13-reverb
Apply a reverb to a dry recording with two contrasting RT60 values. Describe how
the sense of space changes.
```
