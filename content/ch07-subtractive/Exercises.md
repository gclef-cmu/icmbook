# Exercises

```{exercise}
:label: ex-07-sweep
Pass a sawtooth wave through a lowpass filter and sweep the cutoff with an
envelope. Describe the resulting gesture.
```

```{exercise}
:label: ex-07-wind
Using filtered noise, design a "wind" sound. Which filter type and parameters
did you choose, and why?
```

```{exercise}
:label: ex-07-resonance
Increase a filter's resonance until it nearly self-oscillates. What do you hear,
and what does this tell you about resonance?
```
