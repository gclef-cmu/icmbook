# Exercises

```{exercise}
:label: ex-11-melody
Represent a short, familiar melody as a list of (pitch, duration) pairs and
sequence it into sound.
```

```{exercise}
:label: ex-11-rounds
Take the melody from the previous exercise and mix it with a delayed copy of
itself to create a simple round. How long a delay works musically?
```

```{exercise}
:label: ex-11-transform
Write a function that transposes a score by a given number of semitones. Verify
it by ear.
```
