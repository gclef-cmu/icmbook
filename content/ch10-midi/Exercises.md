# Exercises

```{exercise}
:label: ex-10-mtof
Convert MIDI note numbers 60, 69, and 72 to frequencies in hertz using
$f = 440 \cdot 2^{(m-69)/12}$.
```

````{solution} ex-10-mtof
:class: dropdown
60 → 261.63 Hz, 69 → 440 Hz, 72 → 523.25 Hz.
````
