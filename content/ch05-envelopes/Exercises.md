# Exercises

```{exercise}
:label: ex-05-percussive
Design an ADSR envelope for a *percussive* sound (such as a plucked string) and
one for a *sustained* sound (such as a bowed string). Give approximate times for
each stage.
```

```{exercise}
:label: ex-05-clicks
A tone with an instantaneous attack often produces an audible "click". Explain
why, and describe how a short attack ramp fixes it.
```

```{exercise}
:label: ex-05-timbral
Apply an envelope to a parameter *other than* amplitude (for example, filter
cutoff). Describe the effect on the sound.
```
