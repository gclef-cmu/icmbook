# Sampling and Wavetable Synthesis

<!-- ```{image} ../images/chapter08.svg

Instead of computing a waveform from scratch, we can *store* one and read it
back. This chapter covers sample playback and wavetable synthesis.

This chapter covers:

- **Sample playback**: using recorded sound as a source.
- **Wavetables**: efficient oscillators built from stored waveforms.
