# Exercises

```{exercise}
:label: ex-03-install
Install Pyquist following {doc}`Installation`, then run the version check. Record
the version number you see.
```

```{exercise}
:label: ex-03-scale
Starting from the triad example in {ref}`hello-sound`, play an ascending
major scale (eight notes). You may look up the frequencies or compute them.
```

```{exercise}
:label: ex-03-explore
Change one parameter in the first-sound example — frequency, duration, or
amplitude — and describe in one sentence how the result changed.
```
