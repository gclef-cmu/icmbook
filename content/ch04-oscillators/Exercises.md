# Exercises

```{exercise}
:label: ex-04-waveforms
Synthesize a sine, a square, and a sawtooth at the same frequency. Describe how
they differ in sound, and connect each description to its harmonic content.
```

```{exercise}
:label: ex-04-additive
Use additive synthesis to approximate a sawtooth wave with amplitudes
$A_k = 1/k$ for $k = 1 \dots 10$. How does adding more harmonics change the
result?
```

```{exercise}
:label: ex-04-bell
Choose a set of *inharmonic* partials and listen to the result. What kinds of
real instruments produce inharmonic spectra?
```
